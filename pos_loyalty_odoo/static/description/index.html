POS Loyalty Management

This Module is used to give POS loyalty redemption points for every purchase to your customers. Customer can also redeem this loyalty points for other purchase order.

Features

Define Loyalty Points

Define Loyalty based on Pos product category and purchase amount.

Earns loyalty points

Customer can earns loyalty points with his/her every purchase in POS.

Loyalty Point Configuration

Apply loyalty point based on loyalty configuration rule.

Use loyalty points

Customer can use loyalty points as a discount for other purchase order in POS.
