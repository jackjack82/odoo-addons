Sale Order Invoiced Amount

Show amount invoiced and to invoice on Sale Orders!

This module computes for sale order the amount that has been invoiced ad the amount still open. All invoices are included, both draft and validated.

The amount is computed at sale order line level, but it is visible ONLY in debug mode.
